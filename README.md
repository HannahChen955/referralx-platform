# ReferralX

让才华被看见，让连接者成长

## 项目简介

ReferralX是一个公开的招聘推荐平台，旨在帮助更多有才华的人被看见。推荐人通过自己的观察和连接能力，主动促成优秀人才和好机会的相遇。

## 技术栈

- **前端**: Next.js 15 + React 18 + TypeScript
- **样式**: TailwindCSS
- **数据库**: PostgreSQL + Prisma ORM  
- **认证**: NextAuth.js
- **部署**: Vercel (计划)

## 开发环境设置

1. 安装依赖
```bash
npm install
```

2. 设置环境变量
复制 `.env.local` 并填入实际配置

3. 启动开发服务器
```bash
npm run dev
```

4. 打开 [http://localhost:3000](http://localhost:3000)

## 功能特性

- 🔍 职位浏览和搜索
